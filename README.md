# Payroll Calculator (Python)

## Overview
This project is a simple command-line Python application that calculates an employee’s pay based on user input. It generates a basic paystub by computing gross pay, tax deductions, and net pay. This assignment was a part of course CINF108- Programming for Problem Solving. 

## Features
- Accepts employee name and pay information
- Calculates gross pay based on hours worked and hourly rate
- Computes federal and state tax deductions
- Displays total deductions and net pay
- Simple and interactive command-line interface

## Technologies Used
- Python (standard library)

## How to Run
1. Make sure Python is installed on your computer
2. Download or clone this repository
3. Open a terminal in the project folder
4. Run the program:

```bash
python main.py
